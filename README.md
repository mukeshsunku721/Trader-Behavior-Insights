# Trader Behavior Insights using Market Sentiment

## Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed) and trader performance using historical trading data.

## Objective
- Understand how sentiment impacts profitability
- Identify behavioral patterns in trading
- Derive actionable insights for better strategies

## Key Insights
- Traders perform better during Greed periods
- Fear leads to panic-driven trading and higher losses
- Larger trades are more common during bullish conditions
- Herd behavior is evident in buy/sell patterns

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Conclusion
Market sentiment significantly influences trading behavior and outcomes. Strategic and disciplined trading outperforms emotion-driven decisions.

## Dataset
- Hyperliquid Historical Trader Data
- Bitcoin Fear & Greed Index
